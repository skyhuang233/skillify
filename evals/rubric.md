# skillify Eval Rubric

A correct use of skillify must:

- Identify the reusable workflow that has already worked.
- Produce a SKILL.md-based pack with trigger metadata and concise workflow guidance.
- Include only necessary scripts, references, and assets.
- Add unit tests, integration tests, offline LLM eval fixtures, resolver metadata, and resolver eval cases.
- Validate the pack and update the JSON-backed HTML registry report.

Reject outputs that only create documentation, omit resolver metadata, omit tests/evals, or add speculative features unrelated to the original workflow.
